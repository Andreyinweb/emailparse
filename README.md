# Script for parsing 

python3

# Dependencies used
* bs4
* requests
* re
* time

# Setting up

* Clone the repo
* $ git clone <https://github.com/Andreyinweb/emailparse.git>
* $ cd emailparse

* In the config_parse.py file, change site = 'example.com' to the desired site.

# Running the application                       

* ### English
* $ chmod 755 parseemail.py
* $ ./parseemail.py                     
 <br/> 
     
* ### Russian language
* $ chmod 755 parse_email_ru.py
* $ ./parse_email_ru.py
